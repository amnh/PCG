connectedVertex :: VertexLabel -> EdgeInfo -> Maybe VertexLabel Source #

For a given vertex, attempts to get the connected vertex from the EdgeInfo. If the input vertex was present in the EdgeInfo, returns 'Just v' where v is the corresponsing VertexLabel. If the input vertex was not present in the EdgeInfo.

verStreamParser :: (FoldCase (Tokens s), MonadParsec e s m, Token s ~ Char) => m VertexEdgeRoot Source #

Reads two vertex sets and an edge set, conditionally infers the root set when vertex sets are unlabeled. Ensures that the elements of the root set are not connected in the forest. Ensures that the rooted trees in the forest do not contain cycles.
