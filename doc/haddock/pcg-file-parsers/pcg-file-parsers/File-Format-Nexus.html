File.Format.Nexus

Description

The Nexus file format was developed by an committee consisting of members who had no concept of language design. Consequently the Nexus file format is needlessly complex.

This parser works barely on a subset of the Nexus specification's ambiguously defined features.

type AlphabetSymbol = String Source #

Represents a symbol in the character alphabet.

type AmbiguityGroup = [AlphabetSymbol] Source #

Represents an ambiguity group of symbols in the character alphabet. Should always be a non empty list.

type Character = Maybe (Vector AmbiguityGroup) Source #

Character is Maybe, because some chars might not be present for some taxa if unaligned, multiple characters are treated as a single "character", so V.length >= 1 if aligned, each character is a "character", so V.length == 1
