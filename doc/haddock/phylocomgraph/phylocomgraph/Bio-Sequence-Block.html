data CharacterBlock u v w x y z Source #

Represents a block of charcters which are optimized atomically together across networks. The CharacterBlock is polymorphic over static and dynamic character definitions.

Bin types: * u = * v = * w = * x = * y = * z =

Use '(<>)' to construct larger blocks.

Performs a zip over the two character blocks. Uses the input functions to zip the different character types in the character block.

Assumes that the CharacterBlock values have the same number of each character type. If this assumtion is violated, the result will be truncated.
