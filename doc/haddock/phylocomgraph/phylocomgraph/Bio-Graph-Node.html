data SubtreeLeafSet Source #

An arbitrarily ordered collection of leaf nodes in a subtree. Leaves in the tree are uniquely identified by an index across the entire DAG. Set bits represent a leaf uniquily identified by that index being present in the subtree.

Use the Semigroup operation '(<>)' to union the leaves included in a leaf set.
