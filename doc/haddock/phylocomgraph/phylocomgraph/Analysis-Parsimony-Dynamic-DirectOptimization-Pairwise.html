Performs a naive direct optimization. Takes in two characters to run DO on and a metadata object Returns an assignment character, the cost of that assignment, the assignment character with gaps included, the aligned version of the first input character, and the aligned version of the second input character. The process for this algorithm is to generate a traversal matrix, then perform a traceback.
