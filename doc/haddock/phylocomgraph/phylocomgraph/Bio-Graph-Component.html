        class PhylogeneticComponent t i e n | t -> i, t -> n, t -> e where Source #

        Represents the most relaxed phylogentic graph structure.

        The graph must satisfy the following: * The graph is directed * The graph is acyclic * the graph contains one or more root nodes * All nodes have at most in-degree 2 * All nodes have out-degree 0 or out-degree 2

        Represents the most constrained phylogentic graph structure. The constraints correlate to a binary tree.

        The graph must satisfy the following: * The graph is directed * The graph is acyclic * The graph contains exactly one root nodes * All nodes have exactly in-degree 1 * All nodes contain out-degree 0 or out-degree 2
