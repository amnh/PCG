Description

Provides the syntactic combinators for specifying commands conforming to the PCG scripting language syntax. Allows for specifying commands which will be interpreted by the PCG scripting language syntax in a a context-free manner. Being context-free allows for permutations of command component as long as no two command components are ambiguously specified. Such ambiguity may lead to unexpected results when using the provided parser runSyntax.

Commands must be specified using an Applicative style so that a Free Applicative can be used as the syntactic & semantic parser.
